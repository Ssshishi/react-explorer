<h2>🔌 Getting started</h2>
Clone the project:

```bash
$ git clone https://github.com/Goytai/GithubExplorer.git
```

Acess the folder:

```bash
$ cd GithubExplorer
```

Follow the steps below:
```bash
# Install the dependencies
$ yarn

# Start the project
$ yarn start
```
